== Description ==

Music Guru is a child theme of Music Artist WordPress Theme which is best for Music artist, bands, DJs, well, anyone in the music space. Advertise your recent album releases, create an enticing playlist with our best WordPress theme for Music Artist. It comes with features like newsletter options integrated with mailchimp.Also start your website right away with our One Click Demo Import in a minutes.



== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
